Selamat datang di **web-random**! Repository ini berisi kumpulan projek web eksperimental, aplikasi web progresif (PWA), dan *landing page* minimalis yang dirancang untuk berbagai kebutuhan digital.

## 🚀 Daftar Projek & Live Demo

Berikut adalah projek yang tersedia di dalam repository ini. Kamu bisa langsung mencoba versi *live*-nya melalui tautan di bawah:

### 1. 📱 [YouTube PWA](https://kamalcloud52.github.io/web-random/youtube-app)
* **Deskripsi:** Aplikasi YouTube berbasis *Progressive Web App* (PWA) yang dirancang khusus untuk perangkat Android lawas atau berspesifikasi rendah agar tetap bisa menginstal dan menikmati layanan YouTube dengan lancar tanpa aplikasi resmi.
* **Fitur:** Ringan, responsif, dan dapat diinstal langsung ke *homescreen* Android.

### 2. 🎮 [Game Tes Rebahan](https://kamalcloud52.github.io/web-random/game)
* **Deskripsi:** Sebuah *mini-game* berbasis web yang kasual dan minimalis untuk mengukur seberapa tinggi level rebahan atau kegabutanmu. Cocok untuk hiburan singkat di kala senggang.
* **Fitur:** Antarmuka interaktif, animasi *smooth*, dan hasil skor yang instan.

### 3. 📄 [Umbra Landing Page](https://kamalcloud52.github.io/web-random/umbra)
* **Deskripsi:** Halaman pendaratan (*landing page*) modern, bersih, dan estetis yang dibuat sebagai representasi atau media promosi untuk sebuah proyek produk digital bernama **Umbra**.
* **Fitur:** Desain modern, navigasi yang mudah, dan sepenuhnya responsif di semua ukuran layar.

---

## 🛠️ Pengembangan & Teknologi

Semua projek di dalam repository ini dibangun secara mandiri menggunakan teknologi web dasar (*vanilla*) tanpa *framework* berat untuk memastikan performa yang cepat dan optimal:

* **HTML5:** Digunakan untuk menyusun struktur semantik pada setiap halaman web.
* **CSS3:** Memberikan sentuhan visual yang modern, tata letak (*layout*) yang rapi, dan desain yang responsif.
* **JavaScript:** Menangani logika interaktif pada game, fungsionalitas PWA, dan dinamika halaman.
